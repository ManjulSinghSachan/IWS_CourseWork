Create a low-poly effect for your photo

Author: Manjul Sachan

Find a photo with good contrast and brightness.

Remove the background using lasso and pen tools.

Using the brush tool, make triangles on a different layer.

Cover the whole photo with these triangles.

Now open Adobe Illustrator and make a vector mesh using the pen tool and keeping the pre-made triangles as a background.

Select each triangle and then using the dropper tool choose the color inside that triangle and fill it.
